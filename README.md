# ASCII HEADER

Prints the given lines (separatet by ' ') as ascii header (font: Ogre).

## install
`npm install -g mp-ffx/ascii-header`

## usage
`ascii-header foo bar foobarson!`

will print:
```
                        __               
                       / _|  ___    ___  
                      | |_  / _ \  / _ \
                      |  _|| (_) || (_) |
                      |_|   \___/  \___/
                      _                  
                     | |__    __ _  _ __
                     | '_ \  / _` || '__|
                     | |_) || (_| || |   
                     |_.__/  \__,_||_|   
  __                _                                        _
 / _|  ___    ___  | |__    __ _  _ __  ___   ___   _ __    / \
| |_  / _ \  / _ \ | '_ \  / _` || '__|/ __| / _ \ | '_ \  /  /

```
